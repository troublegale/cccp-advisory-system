# Курс B — Введение в компьютерные технологии

## Общая информация

| Параметр | Значение |
|---|---|
| Код курса | B |
| Название | Введение в компьютерные технологии |
| Возраст | 9–11 лет |
| Продолжительность | 96 академических часов |
| Тестирование | Не требуется |

## Для кого

Курс предназначен для школьников 9–11 лет, желающих познакомиться с компьютером как с инструментом, позволяющим эффективно решать разнообразные прикладные задачи.

Курс является **стартовым для всех направлений предпрофессиональной подготовки ДЮКЦ**.

## Требования к поступающим

- **Тестирование:** не требуется.

## Чему научатся учащиеся

- Использовать устройства ввода-вывода информации.
- Анализировать структуру интерфейсов прикладных программ.
- Решать несложные графические задачи в среде FMS-Logo.
- Работать с файловой системой, оптимально организовывать систему хранения цифровых документов.
- Работать с архиваторами.
- Использовать для решения задач сетевые ресурсы и информационные ресурсы ИНТЕРНЕТ.
- Создавать рисунки, текстовые документы, короткие мультфильмы, 3D-модели, одностраничные веб-сайты.

## Изучаемые инструменты и программы

- Операционная система Windows (>8).
- MsPaint.
- Векторный редактор MethodDraw.
- Инструментальная среда FMSLogo.
- Мультипликационный редактор Tupitube.
- Архиватор 7zip.
- 3D-редактор SculptGL.
- Конструктор сайтов Nethouse.
