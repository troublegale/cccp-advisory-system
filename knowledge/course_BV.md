# Курс BV — Введение в компьютерную графику

## Общая информация

| Параметр | Значение |
|---|---|
| Код курса | BV |
| Название | Введение в компьютерную графику |
| Возраст | 10–11 лет |
| Продолжительность | 96 академических часов |
| Тестирование | Зависит от категории поступающего (см. ниже) |

## Для кого

Курс предназначен для школьников 10–11 лет, желающих познакомиться с различными областями применения компьютерной графики, в том числе — с использованием компьютерной графики для создания простых компьютерных игр.

Курс является **этапом предпрофессиональной подготовки ДЮКЦ** по направлениям:

- Графический дизайн.
- 2D-мультипликация и разработка 2D игр.
- 3D-моделирование и разработка 3D игр.
- Web-разработка.
- Информационная безопасность.
- Конструирование и робототехника.

## Требования к поступающим

- **Для учащихся ДЮКЦ:** тестирование не требуется.
- **Для вновь поступающих:** тестирование по курсу «B» («Введение в компьютерные технологии»).

## Чему научатся учащиеся

- Создавать несложные векторные и растровые изображения (в т.ч. иллюстрации).
- Использовать графические редакторы для создания фона и персонажей простых компьютерных игр.
- Делать gif-анимации.
- Создавать 3D-модели.
- Разрабатывать простые компьютерные игры.

## Изучаемые инструменты и программы

- Растровый редактор Gimp.
- Фоторедактор Pixlr.
- Редактор для цифровой живописи ArtRage.
- Векторный редактор InkScape.
- Среда программирования и редактор Scratch.
- 3D-редактор Solid Kids.
- Среда для разработки презентаций MS PowerPoint.
